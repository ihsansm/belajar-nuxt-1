# Nuxt 

[Nuxt documentation](https://nuxt.com/docs/getting-started/introduction).

## Setup

dependencies:

```bash
# npm
npm install
```

## Development Server

`http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```

[deployment documentation](https://nuxt.com/docs/getting-started/deployment).
